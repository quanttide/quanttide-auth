# quanttide-auth
量潮身份认证

## 概述

量潮身份认证（quanttide-auth）是量潮知识管理体系中的**身份认证与权限管理**领域，涵盖统一身份管理、认证协议、授权模型、权限策略等核心能力。

## 领域边界

- **身份生命周期管理**：用户、组织、角色的创建、同步、注销
- **认证协议与实现**：OAuth 2.0、OIDC、SAML、LDAP 等标准协议
- **授权与权限模型**：RBAC、ABAC、ReBAC 等权限模型
- **安全策略**：多因素认证、会话管理、令牌管理
